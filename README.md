# Physics #

Simulations of some basic (and a few advanced) concepts in physics.

Topics include:
* Motion and Kinematics
  * Free Fall
  * Projectile Motion
  * Air Resistance
* Force and Friction
  * Newton's Cradle
  * Box on a Ramp
  * Pulley System
* Circular Motion
  * Car on a Track
  * Torque
  * Gravity/Orbit
* Momentum
  * Car Collision
* Work and Energy
  * Springs
  * Inelastic Collision
* Oscillations
  * Pendulum/double Pendulum
  * Wave on a String
* Electricity and Magnetism
* Optics
